# NayePankh Foundation Website

🚀 **A modern, responsive, animated single‑page website** showcasing the mission, programs, and impact of the NayePankh Foundation.

---

## 🎯 Project Overview

**NayePankh Foundation Website** is a lightweight, accessible, and visually engaging web page built with vanilla HTML5, CSS3, and JavaScript. It demonstrates a clean mobile‑first design, smooth navigation, subtle animations, and a theme‑aware dark‑mode toggle.

---

## ✨ Key Features

- **Responsive Design** – Optimised for mobile, tablet, and desktop using Flexbox and CSS Grid.
- **Smooth Scroll Navigation** – Header links scroll gracefully to each section.
- **Subtle Animations** – Fade‑in text, hover effects on buttons/cards, and scroll‑triggered transitions.
- **Dark‑Mode Toggle** – Seamless colour switching with persisted user preference.
- **Multi‑Section Layout** – Hero, About, Programs/Initiatives, Gallery/Impact, and Contact sections.
- **Performance‑Focused** – No external frameworks; only native web technologies.

---

## 🛠️ Tech Stack

- **HTML5** – Semantic markup for accessibility and SEO.
- **CSS3** – Custom properties, Flexbox/Grid, media queries, and animations.
- **JavaScript** – Dark‑mode handling, IntersectionObserver for fade‑ins, and smooth scrolling.
- **Design Principles** – Light‑weight, accessible, and follows modern UI/UX best practices.

## 📦 Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/naye-pankh-website.git
   cd naye-pankh-website
   ```
2. **Open the site**
   - Simply double‑click `index.html` or open it in any browser.
3. **Optional – Serve locally** (useful for live‑reload)
   ```bash
   npx -y live-server@latest .
   ```

---

## ▶️ Usage

- **Navigate** – Click the navigation links in the header to jump to sections.
- **Toggle Dark Mode** – Click the 🌙 button in the header; the preference is saved in `localStorage`.
- **Explore Animations** – Scroll down to watch sections fade in and hover over cards/buttons for interactive effects.

---

## 🌐 Deployment

Deploy to any static‑site host (GitHub Pages, Vercel, Netlify, etc.). Example for GitHub Pages:
1. Push the code to a GitHub repository.
2. In repository settings, enable **GitHub Pages** on the `main` branch (or `docs/` folder).
3. Your site will be live at `https://your-username.github.io/naye-pankh-website/`.
